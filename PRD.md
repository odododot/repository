# PRD: StockView (주식 데이터 시각화 앱)

## 1. Introduction
StockView는 사용자에게 코인 매수 추천을 제공하는 애플리케이션입니다. 특정 기술적 분석 전략을 기반으로 그래프에 현재 가격과 추천 지점을 표시하고, 그에 대한 명확한 이유를 제시하여 사용자의 투자 결정을 돕습니다.

## 2. Goals
- 사용자가 기술적 분석에 기반한 매수 시점을 쉽게 파악할 수 있도록 돕습니다.
- 복잡한 차트 분석 없이도 직관적인 시각 자료와 근거를 통해 투자 정보를 얻을 수 있는 환경을 제공합니다.
- 안정적이고 확장 가능한 애플리케이션 아키텍처를 구축합니다.

## 3. Target Audience
- 기술적 분석에 익숙하지 않은 초보 코인 투자자
- 특정 매매 전략(20일 이동평균선, 상승 다이버전스)을 활용하여 투자 아이디어를 얻고 싶은 중급 투자자
- 모바일 환경에서 빠르고 편리하게 코인 매수 추천 정보를 확인하고 싶은 사용자

## 4. Features
- **코인 가격 시각화 그래프**
    - 현재 코인 가격을 그래프 형태로 시각화하여 보여줍니다.
    - 그래프 상에 20일 이동평균선을 함께 표시합니다.
    - 상승 다이버전스 발생 시 이를 그래프에 표시합니다.
- **매수 추천 및 근거 제시**
    - 정의된 전략(20일 이동평균선 돌파, 상승 다이버전스 발생)에 부합하는 시점이 오면 매수를 추천합니다.
    - 그래프 하단에 어떤 전략에 의해 매수 추천이 발생했는지에 대한 구체적인 이유를 텍스트로 설명합니다.

## 5. Technical Requirements
- **Platform:** Flutter
- **Architecture:**
    - Clean Architecture
    - MVVM (Model-View-ViewModel) 패턴
    - SOLID 원칙 준수
- **State Management:**
    - 외부 라이브러리 사용 금지
    - `ChangeNotifier`와 `ListenableBuilder`를 조합하여 상태 관리 구현
    - State Holder 패턴 적용
- **Data:**
    - API 키는 `.env` 파일을 통해 관리
    - 테스트용 가짜 데이터(Mock Data)를 활용
    - 의존성 주입(DI)을 통해 테스트용 데이터와 실제 데이터를 쉽게 교체할 수 있는 구조 구현
    - **데이터 저장:** 로컬 저장소만 사용
- **Testing:**
    - 데이터 로직의 정확성을 검증하기 위한 유닛 테스트 작성

## 6. Out of Scope
- 실제 매매 기능 (주문, 체결 등)
- 사용자 계정 및 인증 시스템 (인증 기능 없음)
- 20일 이동평균선, 상승 다이버전스 외 다른 매매 전략
- 코인 매도 추천 기능

## 7. Success Metrics
- 사용자가 매수 추천의 근거를 명확하게 이해하고 유용하다고 평가함
- 데이터 로딩 및 그래프 렌더링이 안정적이고 빠르게 동작함
- 유닛 테스트를 통해 데이터 처리 로직의 신뢰성을 확보함
