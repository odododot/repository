# StockView(주식 데이터 시각화 앱)
- 코인 매수 추천 앱을 만들어줘.
- 전략을 활용하여 그래프에서 현재 가격과 전략을 표시하고, 하단에 이유를 작성해줘.
- 전략 : 20일 이동평균선, 상승 다이버전스

# 기술 스택
- Flutter
- MVVM 패턴
- 상태관리는 라이브러리 사용 금지, ChangeNotifier + ListenableBuilder 조합으로 할 것

# 화면 구성
- 상단에 그래프, 하단에 추천 이유
  
# Notes
- API_KEY : .env
- State Holder 패턴을 사용
- 테스트용 가짜 데이터 활용
- 테스트용 데이터와 실제 데이터 간편하게 swap 가능한 구조(DI)
- 데이터가 잘 가져와 지는지 유닛 테스트
- 클린 아키텍처 사용
- SOLID
